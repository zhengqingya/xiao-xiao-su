### 一、前言

最近整合`Spring Boot`+`Spring Security`+`JWT`+`Vue` 完成了一套前后端分离的基础项目，这里把它开源出来分享给有需要的小伙伴们

功能很简单，单点登录，前后端动态权限配置，前端权限精确到 `按钮` 级别，后端权限精确到 `url` 上，剩下的就是一些关联表的增删改查，例如：用户管理，角色管理，菜单管理，系统日志等...
